# Semana 7 - Constructores y Destructores en Python

## Descripción
Programa simple de gestión de cuentas bancarias que demuestra:
- Uso de `__init__` para inicializar atributos y abrir recursos (archivo log)
- Uso de `__del__` para cerrar recursos y registrar destrucción

## Estructura
- `modelos/` → entidades (CuentaBancaria)
- `servicios/` → lógica de negocio (GestorCuentas)
- `main.py` → punto de entrada

## Cómo ejecutar
```bash
python main.py
